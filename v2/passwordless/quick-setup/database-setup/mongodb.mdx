# MongoDB setup

:::important
- This is needed only if you are running the SuperTokens core yourself.
- We support only session management with MongoDB. Login support will be added in the future.
:::

## 1) Connect SuperTokens to your database ðŸ”Œ

<SelfHostedTabs>
<TabItem value="with-docker">

```bash
docker run \
	-p 3567:3567 \
    // highlight-next-line
    -e MONGODB_CONNECTION_URI="mongodb://root:root@192.168.1.2:27017/dbName" \ 
    -d registry.supertokens.io/supertokens/supertokens-mongodb
```
</TabItem>

<TabItem value="without-docker">

```yaml
# You need to add the following to the config.yaml file.
# The file path can be found by running the "supertokens --help" command

mongodb_connection_uri: "mongodb://root:root@192.168.1.2:27017/dbName"
```
</TabItem>
</SelfHostedTabs>
