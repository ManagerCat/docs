# EmailPassword and Social / Third Party login with SuperTokens

## Features

✅ Sign-up / Sign-in with email ID and password <br/>
✅ Sign-up / Sign-in with third party providers (Like google, facebook etc..) <br/>
✅ Forgot password flow using email <br/>
✅ Secure session management <br/>
✅ Email verification <br/>

## Demo app

See our [live demo app](https://thirdpartyemailpassword.demo.supertokens.com/) or view the [React + NodeJS source code on Github](https://github.com/supertokens/supertokens-auth-react/tree/master/examples/with-thirdpartyemailpassword)
