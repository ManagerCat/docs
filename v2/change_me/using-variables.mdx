## In Header elements

Config

```json
{
    "change_me": {
        "headerVariable": "Variable in header"
    }
}
```

Usage

```markdown
# H4: ^{variableName}
```

Sample

#### H4: ^{headerVariable}

## In code blocks

Config

```json
{
    "change_me": {
        "codeBlockContent": "Variable value",
        "codeBlockTitle": "Custom Code Block Title"
    }
}
```

Usage
    ```markdown
    var someVariable = "^{variableName}"
    ```

    ```markdown title="^{variableName}"
    let test = "Test"
    ```

Sample
```tsx
var someVariable = "^{codeBlockContent}"
```

```tsx title="^{codeBlockTitle}""
let test = "Test"
```

## In Admonitions

Config

```json
{
    "change_me": {
        "admonitionText": "This is text inside an admonition"
    }
}
```

Usage

    :::important
    ^{variableName}
    :::

Sample

:::important
^{admonitionText}
:::

## In normal markdown elements

```json
{
    "change_me": {
        "markdownText": "Some markdown text",
        "markdownPartialLinkText": "somemarkdownlink",
        "markdownLink": "https://supertokens.com",
        "markdownBullet": "Markdown bullet"
    }
}
```

```markdown
[Link1](^{markdownLink})
[Link2](https://somelink/^{markdownPartialLinkText})

In normal text: ^{markdownText}

- In bullet points
- ^{markdownBullet}
```

## In React Components

```json
{
    "change_me": {
        "reactTabContent": "Tab value",
        "reactTabsDefaultValue": "orange"
    }
}
```

Usage

    ```tsx
    // As content for a component
    <Tabs
        groupId="fruits"  
        defaultValue="apple"
        values={[
        {label: 'Apple 2', value: 'apple'},
        {label: 'Orange 2', value: 'orange'},
        {label: 'Banana 2', value: 'banana'},
        ]}>
        <TabItem value="apple">^{variableName} üçé</TabItem>
        <TabItem value="orange">This is an orange üçä</TabItem>
        <TabItem value="banana">This is a banana üçå</TabItem>
    </Tabs>

    // As props for a component
    <Tabs
        groupId="fruits"  
        defaultValue="^{variableName}"
        values={[
        {label: 'Apple 2', value: 'apple'},
        {label: 'Orange 2', value: 'orange'},

    ```
